## 简介

HealthTick 是一款 macOS 菜单栏久坐提醒工具，帮助你在长时间工作时定期休息，养成健康习惯。它轻量常驻于菜单栏，不打扰工作，同时通过打卡、徽章、统计等功能激励你坚持休息。

**系统要求：** macOS 14 (Sonoma) 或更高版本，支持 Apple Silicon 和 Intel。

---

## 安装与启动

### 方式一：Homebrew（推荐）

```bash
brew tap lifedever/tap
brew install --cask health-tick
```

升级：
```bash
brew upgrade --cask health-tick
```

### 方式二：手动下载

从 [GitHub Releases](https://github.com/lifedever/health-tick-release/releases/latest) 下载 `.dmg` 文件，打开后将 HealthTick 拖入 Applications 文件夹。

> **首次打开提示**：如果系统提示"无法验证开发者"，前往 **系统设置 → 隐私与安全性**，点击"仍要打开"。

### 开机自启

前往**设置 → 系统**，开启"登录时启动"，让 HealthTick 随系统自动运行。

## 菜单栏界面

点击菜单栏中的 HealthTick 图标，展开主界面，包含以下区域：

### 顶部状态提示

| 提示 | 含义 |
|------|------|
| 绿色"今日目标已达成" | 当天休息次数已达到设定目标 |
| 橙色"已跳过 3 次警告" | 今日跳过休息次数过多，提示注意健康 |
| 紫色"免打扰模式" | 当前处于安静时段或工作时间范围外，计时已暂停 |

### 计时圆圈

- 圆形进度条显示当前工作时间的剩余进度
- 中央数字为剩余时间（分:秒格式）
- **颜色含义：**
  - 绿色：正在计时（工作中）
  - 橙色：已暂停 / 免打扰模式
  - 红色：休息提醒弹出中
  - 蓝色：等待确认开始休息
- 工作状态下，圆圈中央有"手动休息"按钮，可随时提前触发休息

### 今日数据

| 数字 | 含义 |
|------|------|
| 已完成 | 今天已完成的休息次数 |
| 目标 | 每日目标休息次数 |
| 连续 | 当前连续达标天数（连续打卡） |
| 跳过 | 今日跳过休息次数（出现时显示） |

下方还显示今日工作时长，以及最近 7 天的完成情况色块（绿色越深表示完成越多）。

### 徽章提示

如果你距离下一枚徽章很近，此处会提示还差几天可以解锁；已获得徽章时则展示最新徽章。

### 控制按钮

| 按钮 | 功能 |
|------|------|
| 暂停 / 继续 | 暂停或恢复计时器 |
| 重置 | 重置当前计时周期 |
| 成就 | 打开统计与徽章窗口 |
| 帮助 | 打开使用引导 |
| 设置 | 打开设置窗口 |

## 计时器工作流程

HealthTick 使用工作-休息循环模式：

```
工作计时 → 休息提醒 → 休息中 → 工作计时 → ...
```

1. **工作阶段**：计时器倒计时（默认 60 分钟），期间可以专心工作。
2. **提醒阶段**：工作时间结束，弹出休息提醒。
3. **休息阶段**：开始休息倒计时（默认 2 分钟），期间检测到键鼠操作会暂停倒计时，确保真正休息。
4. **循环继续**：休息结束后自动开启下一轮工作计时。

### 操作检测

休息期间，HealthTick 会检测键盘和鼠标操作。若检测到操作，休息倒计时将暂停，提示你停止操作、真正休息。

## 休息提醒

### 提醒位置

可在**设置 → 休息**中选择休息弹窗的显示位置：

| 位置 | 说明 |
|------|------|
| 菜单栏窗口 | 在菜单栏弹出窗口内显示，不覆盖屏幕 |
| 右上角 | 屏幕右上角浮窗 |
| 左上角 | 屏幕左上角浮窗 |
| 屏幕中央 | 屏幕正中央浮窗 |
| 全屏强制 | 全屏遮罩，强制休息 |

### 休息活动建议

每次休息弹窗会随机显示一条活动建议，包括：

- 起来走走，活动身体
- 远眺窗外，放松眼睛
- 喝杯水，补充水分
- 做简单拉伸动作
- 深呼吸，放松身心
- 活动手腕，预防鼠标手
- 闭眼休息，让大脑放松
- 伸展脊柱，改善坐姿

### 跳过与确认

- **跳过**：若允许跳过（设置中可配置），点击跳过按钮可结束休息提醒并继续工作。跳过次数会被记录，当天跳过 3 次或以上时会在菜单栏显示警告。
- **确认后休息**：若开启"确认后休息"选项，提醒弹出时需要手动点击"开始休息"才进入休息倒计时。

### 手动休息

工作中随时可以点击菜单栏圆圈中的"手动休息"按钮，提前触发休息。

## 每日目标与打卡

### 每日目标

在**设置 → 应用**中设置每天的目标休息次数（1-20 次，默认 8 次）。

- 当天完成的休息次数达到目标后，菜单栏会显示绿色"今日目标已达成"提示
- 可开启"达成目标后自动暂停"，让计时器在目标达成后停止，避免过度提醒

### 连续打卡

每天完成目标即算打卡成功，连续打卡天数会在菜单栏界面显示为"连续 N 天"。

- 工作日设定：可在设置中选择哪些星期几算工作日，非工作日不计入连续打卡考核
- 连续打卡是解锁连续打卡徽章的关键指标

## 徽章系统

HealthTick 提供两类徽章，共 18+ 枚：

### 连续打卡徽章

基于**最长连续打卡天数**解锁，从"初出茅庐（3天）"到"传奇（365天）"逐级解锁。

### 累计打卡徽章

基于**累计总休息次数**解锁，从"50次里程碑"到"5000次殿堂"逐级解锁。

### 查看徽章

点击菜单栏界面的"成就"按钮，打开成就窗口，切换到"徽章"标签页：

- 已解锁的徽章显示为彩色，带金色圆环
- 未解锁的徽章显示为灰色，带进度弧线显示当前进展
- 点击已解锁的徽章会触发庆祝动画
- 底部显示距离下一枚徽章还差多少，附进度条

### 分享

在菜单栏界面点击分享按钮（工作时长旁边的上传图标），可以生成打卡卡片并分享。

## 数据统计

点击菜单栏界面"成就"按钮，打开成就窗口，"统计"标签页展示以下数据：

### 数据卡片（8 块）

| 卡片 | 内容 |
|------|------|
| 今日完成 | 当天完成次数 / 目标次数 |
| 当前连续 | 当前连续打卡天数 |
| 最长连续 | 历史最长连续记录 |
| 累计总次 | 所有历史休息次数之和 |
| 今日工作 | 今天的有效工作时长 |
| 本周达标 | 本周达标天数 / 本周工作日数 |
| 本月达标 | 本月达标天数 / 本月工作日数 |
| 最佳单日 | 历史单日最高休息次数 |

### 近 7 天柱状图

展示最近 7 天每天的休息次数，绿色柱子越高表示完成越多，达到目标的天显示深绿。

### 概览面板

- 使用天数：从首次记录到今天的天数
- 活跃天数：至少完成一次休息的天数
- 日均次数：平均每天完成的休息次数
- 最佳单日：最高记录及日期
- 今日跳过：当天跳过休息次数（有跳过时显示）

### 近 30 天热力图 / 工作时长图

底部区域支持两种视图切换：

- **打卡次数**：30 天热力图，颜色越深代表次数越多
- **工作时长**：30 天柱状图，点击某根柱子可查看该天的具体工作时长

## 设置详解

点击菜单栏界面"设置"按钮，打开设置窗口，共 5 个标签页。

### 系统标签页

| 选项 | 说明 |
|------|------|
| 语言 | 切换应用语言（跟随系统 / 中文 / English） |
| 外观 | 切换应用主题（跟随系统 / 浅色 / 深色） |
| 登录时启动 | 开机自动启动 HealthTick |
| 重新打开引导 | 重新播放首次使用引导 |
| 重置设置 | 将所有设置恢复为默认值 |
| 导出数据 | 将历史数据加密导出为 `.htdata` 文件 |
| 清除所有数据 | 删除全部历史记录（不可恢复） |

### 应用标签页

| 选项 | 说明 |
|------|------|
| 工作时长 | 每次工作周期的时长（1-120 分钟，默认 60 分钟） |
| 休息时长 | 每次休息的时长（20 秒-15 分钟，默认 2 分钟） |
| 每日目标 | 每天目标休息次数（1-20 次，默认 8 次） |
| 长休息 | 每完成 N 个周期触发一次更长的休息（5-30 分钟） |
| 护眼模式 | 自动设为 20-20 规则：每 20 分钟休息 20 秒 |
| 工作日 | 选择哪些天算工作日（影响打卡考核） |
| 工作时段 | 设定每天工作的时间范围，范围外自动暂停提醒 |
| 安静时段 | 设定免打扰时间段（可多条，支持按星期过滤） |
| 达成目标后自动暂停 | 目标完成后停止计时，不再触发休息 |

#### 护眼模式

开启后，工作时长自动设为 20 分钟，休息时长自动设为 20 秒，符合眼科推荐的 20-20-20 护眼规则。关闭后恢复之前的设置。

#### 长休息

每完成指定数量的短休息周期（默认 4 个）后，自动触发一次更长的休息（默认 15 分钟）。适合需要深度工作的用户。

#### 安静时段

可添加多个免打扰时间段（如午休 12:00-13:00，可以指定仅在某些星期生效）。在安静时段内，计时器暂停，不会弹出休息提醒。安静时段结束后，计时器自动恢复。

### 休息标签页

| 选项 | 说明 |
|------|------|
| 提醒位置 | 休息弹窗出现的位置（菜单栏 / 右上角 / 左上角 / 中央 / 全屏） |
| 确认后休息 | 开启后，需要手动点击"开始休息"才进入休息倒计时 |
| 允许跳过 | 开启后，休息提醒中显示跳过按钮 |
| 提示音 | 选择休息开始时的提示音（14 种系统音效） |
| 操作检测提示音 | 休息中检测到操作时播放的提示音 |

### 提醒文字标签页

可以自定义休息弹窗中显示的提醒语，最多 2 条，支持随机轮换显示。

### 关于标签页

显示当前版本号、更新日志入口和联系方式。

## 数据管理

### 数据导出

前往**设置 → 系统 → 导出数据**：

1. 输入自定义密码（用于加密保护）
2. 选择保存路径
3. 文件以 `.htdata` 格式保存

导出的文件包含所有历史记录，文件已加密，需要密码才能读取。

### 清除数据

前往**设置 → 系统 → 清除所有数据**，二次确认后清除全部历史记录。**此操作不可恢复，请谨慎操作。**

### 自动更新

HealthTick 启动时会自动检查新版本。有可用更新时，菜单栏界面会显示更新提示，点击即可跳转下载。

## 常见问题

**Q：首次打开提示"无法验证开发者"？**  
前往**系统设置 → 隐私与安全性**，找到 HealthTick 相关提示，点击"仍要打开"。

**Q：休息期间为什么倒计时停了？**  
HealthTick 检测到你有键盘或鼠标操作，倒计时暂停，提示你真正放松。停止操作后倒计时会继续。

**Q：如何临时关掉提醒但不退出应用？**  
可以点击菜单栏界面的"暂停"按钮暂停计时器；或在设置中添加一个安静时段覆盖当前时间段。

**Q：设置了工作时段，但下班后仍然有提醒？**  
请确认在**设置 → 应用**中开启了"工作时段"开关，并正确设置了开始和结束时间。

**Q：连续打卡天数为什么没有增加？**  
连续打卡要求当天完成的休息次数达到"每日目标"设定值。另外，如果该天不在你设定的"工作日"范围内，也不计入连续考核。
